This package provides a set of lints for the Velvet projects.

## Installation

Add the following to your `pubspec.yaml`:

```yaml
dev_dependencies:
  velvet_lints: ^0.1.0
```

## Usage

Add the following to your `analysis_options.yaml`:

```yaml
include: package:velvet_lints/flutter.yaml
```
